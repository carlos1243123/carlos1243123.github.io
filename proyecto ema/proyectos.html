Proyecto: Calculadora de Figuras Geométricas y Detector de Palíndromos

Descripción General

Este proyecto consiste en desarrollar un programa en lenguaje C que:

  1. Calcula el área y perímetro de figuras geométricas básicas (cuadrado, rectángulo, círculo, triángulo).
  2. Determina si una cadena de texto es un palíndromo.

Funcionalidades

Calculadora de Figuras Geométricas

  1. Menú para seleccionar la figura geométrica.
  2. Ingreso de datos para calcular área y perímetro.
  3. Visualización de resultados.

Formulario de Veterinaria en JavaScript

Descripción

Este proyecto consiste en desarrollar un formulario web en JavaScript que permita a los dueños de mascotas registrar información sobre sus animales y solicitar servicios veterinarios.

Funcionalidades

  1. Registro de información de la mascota (nombre, especie, raza, edad, etc.).
  2. Selección de servicios veterinarios (vacunación, esterilización, consulta, etc.).
  3. Asignación de fecha y hora para la cita.
  4. Validación de campos requeridos.
  5. Envío de información a un correo electrónico o base de datos.

Cuestionario de Clash Royale en C++

Descripción

Este proyecto consiste en desarrollar un programa en lenguaje C++ que simula un cuestionario sobre el juego popular Clash Royale. El programa presentará una serie de preguntas relacionadas con el juego y evaluará las respuestas del usuario.

Funcionalidades

  1. Presentación de preguntas: El programa mostrará una serie de preguntas relacionadas con Clash Royale.
  2. Evaluación de respuestas: El programa evaluará las respuestas del usuario y asignará puntos según su precisión.
  3. Clasificación: El programa mostrará la clasificación del usuario al final del cuestionario.
  4. Opción de repetir: El programa dará la opción de repetir el cuestionario.
